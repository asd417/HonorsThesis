\section{PSPACE-complete}

On top of the Sokoban being an NP-hard game, Sokoban was also found to be PSPACE-complete. A PSPace problem is a decision problem that can be solved by a Turing machine using a polynomial amount of memory space. If every problem P in PSPACE can be reduced to a problem, the problem is PSPACE-hard. A problem is PSPACE-complete if and only if it is a PSPACE and a PSPACE-hard problem.
